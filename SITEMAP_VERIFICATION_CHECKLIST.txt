SITEMAP VERIFICATION CHECKLIST
================================

✅ SITEMAP URL TO CHECK:
https://taxiparis.org/sitemap.xml

WHAT YOU SHOULD SEE:
===================

When you visit https://taxiparis.org/sitemap.xml in your browser, you should see:

1. XML Structure:
   - <?xml version="1.0" encoding="UTF-8"?>
   - <urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
   - Multiple <url> entries for different taxi services

2. Key Pages in Sitemap:
   - Homepage (priority 1.0)
   - Airport transfers (CDG, Orly, Le Bourget) - high priority
   - Train stations (Gare du Nord, Lyon, Montparnasse)
   - 24h taxi service
   - VTC Paris services
   - General Île-de-France service pages

3. Each URL should have:
   - <loc> (location/URL)
   - <lastmod> (last modified date: 2025-11-12)
   - <changefreq> (update frequency)
   - <priority> (importance: 0.3 to 1.0)

GOOGLE SEARCH CONSOLE STEPS:
============================

1. SUBMIT SITEMAP:
   - Go to Google Search Console
   - Select your taxiparis.org property
   - Click "Sitemaps" in left menu
   - Click "Add a new sitemap"
   - Enter: sitemap.xml
   - Click "Submit"

2. VERIFY SUBMISSION:
   - Sitemap should appear in the list
   - Status: "Success" (green checkmark)
   - Should show number of URLs discovered
   - No errors or warnings

3. CHECK COVERAGE:
   - Go to "Coverage" in left menu
   - See how many pages are indexed
   - Check for any crawl errors

EXPECTED RESULTS:

✅ All main pages indexed

COMMON ISSUES TO WATCH FOR:
==========================

❌ "Couldn't fetch" - Server issue, check if site is accessible
❌ "Couldn't parse" - XML syntax error (shouldn't happen with our file)
❌ "Invalid URL" - URL format issues
❌ "HTTP Error" - Server configuration problems
